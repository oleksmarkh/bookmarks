## Notes

Legend (categories):

* :blue_book: - docs
* :books: - book
* :notebook: - blog
* :memo: - blog post, tutorial, guide
* :microphone: - conference, meetup
* :scroll: - presentation
* :movie_camera: - video
* :radio: - podcast
* :newspaper: - media
* :link: - catalog, digest
* :speech_balloon: - community forum
* :eyeglasses: - style guide
* :wrench: - tool (lib/framework)
* :star: - favorite

When adding a new link, make sure it's ordered according to this categorization (also applies to sublists).
